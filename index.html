Security ratings
bcrypt
Adaptive cost factor (rounds=12). Salted automatically. Slows down as hardware improves. Used in this project's SecureAuth.
Recommended
Argon2id
Memory-hard. PHC winner 2015. Resists GPU + ASIC attacks. Best choice for new systems. More configurable than bcrypt.
Recommended
SHA-256
Designed for speed. No cost factor, no built-in salt. Billions of hashes/sec on GPUs. Used in this project only for demonstration.
Insecure for passwords
MD5
Cryptographically broken. Used in generate_test_data.py and attack scripts intentionally to show vulnerability. Never use in production.
Deprecated / Broken
